# 🍰 Cantinho Doce - Catálogo Digital & E-commerce

Catálogo digital interativo e responsivo desenvolvido especificamente para o nicho de confeitaria, docerias artesanais e kits festa. O sistema atua como um mini e-commerce front-end, focando na experiência do usuário e na conversão direta de vendas via WhatsApp.

🔗 **[Acessar a Demonstração Online](https://tinyurl.com/estilo-doce)**

## 🚀 Funcionalidades Principais

* **Carrinho de Compras Dinâmico:** Sistema flutuante que soma automaticamente os valores e organiza os itens escolhidos pelo cliente.
* **Checkout no WhatsApp:** Envio do pedido completamente formatado para o número do estabelecimento, incluindo descrição dos itens, valores unitários e valor total da compra.
* **Módulo "Monte seu Cento":** Sistema interativo oculto (Toggle) que permite ao cliente escolher até 4 sabores de salgadinhos, com validação inteligente que exige o fechamento exato de 100 unidades antes de permitir a adição ao carrinho.
* **Design Responsivo (Mobile-First):** Layout adaptável utilizando CSS Grid e Flexbox, garantindo a mesma qualidade de navegação tanto em smartphones quanto em desktops.
* **Performance:** Desenvolvido sem frameworks pesados, garantindo carregamento instantâneo.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização, animações (hover, pulse) e Media Queries para responsividade.
* **JavaScript (Vanilla):** Lógica do carrinho de compras, manipulação do DOM e integração de links dinâmicos para o WhatsApp.
